# Texto da Proposta Comercial — Estrutura Persuasiva

## Filosofia de copy

A proposta é entregue ao **cliente final** do parceiro. Ela precisa:
1. Legitimar o parceiro como especialista (não apenas vendedor)
2. Ancorar o valor antes de mostrar o preço
3. Tornar o "não contratar" mais caro cognitivamente do que contratar
4. Terminar com clareza total sobre o próximo passo

## HIERARQUIA VISUAL DO PDF (para o desenvolvedor)

```
1. CAPA (header gradient escuro → cor da marca)
   ├── Logo
   ├── "PROPOSTA COMERCIAL PERSONALIZADA"
   └── Nome da empresa + data

2. PROBLEMA (fundo branco, texto emocional, fonte grande)

3. O QUE VOCÊ TEM EM MÃOS (cards de infraestrutura)
   └── Ícone + Nome + Linha de impacto + Quantidade

4. RECURSOS INCLUÍDOS (chips/badges coloridos com cor da marca)
   └── Nome do módulo + Linha de impacto

5. INVESTIMENTO (caixa escura, preço em destaque verde)
   ├── Texto de ancoragem
   ├── Preço mensal (fonte grande, verde)
   └── Taxa de ativação (se houver)

6. POR QUE AGIR AGORA (fundo suave cor da marca, call-to-action)

7. SOBRE O EMISSOR (neutro)

8. FOOTER (logo + contato + data)
```

## REGRAS DE COPY

1. **Nunca mencionar**: comissão, tier, parceiro, acréscimo, programa de revendedores
2. **Sempre personalizar**: nome da empresa do cliente no header e no texto de abertura
3. **Âncora de valor antes do preço**: o leitor deve sentir que o preço é barato antes de vê-lo
4. **Uma CTA clara**: "Entre em contato" ou "Assine agora" — nunca duas CTAs simultâneas
5. **Textos curtos por bloco**: máximo 3 linhas por parágrafo no PDF
6. **Cores**: usar `colorBrandPrimary` para destaques, `colorPartner` APENAS se o negócio é do universo parceiro (não usar para o cliente final)
